# AI lead 

## Описание

AI Product Lead (MWS) — Observability, LLM/RAG, Multi‑Agent Systems
- Развиваю внутренние AI‑продукты для инженеров и эксплуатации: LLM/RAG‑ассистенты, мультиагентные системы и предиктивная аналитика, встроенные в контур Observability для mission‑critical продуктов ГК МТС. 
- Отвечаю за value продукта (метрики/внедрение), стратегию (роадмап/бэклог), а также за delivery и команду.

Ключевые достижения:
- Запустил RAG-приложение для поиска по документации и БД: MAU 1200, единая точка доступа к знаниям и данным.
- Развернул OPS Find — мультиагентный хаб (LLM/ML + A2A/MCP) для поиска данных и выполнения действий во внутренних сервисах (self‑service для инженеров).
- Внедрил AI‑помощника инженера по инцидентам (обогащение контекстом из мониторинга): ‑10% времени решения инцидента.
- Автоматизировал ML‑маршрутизацию инцидентов (scikit‑learn): ‑22% времени маршрутизации, повышение точности назначения команд поддержки.
- Построил предиктивный анализ для 3000+ серверов геораспределенных PostgreSQL (Prophet): профилактика аварий и раннее выявление деградаций.
- Запустил централизованный мониторинг корпоративных AI‑агентов на LangFuse: контроль нагрузки, стоимости и качества ответов (foundation для масштабирования AI‑продуктов).


AI/LLM продукты, которые довел до внедрения:
- Grafana Assistant: анализ метрик, помощь в построении дашбордов.
- Агент диагностики ИТ‑продукта: анализ логов/трейсов/метрик/событий и поиск аномалий.
- Solution & Postmortem Copilot (multi‑agent): поиск похожих инцидентов, генерация гипотез причин, мер предотвращения повторов.
- Карта взаимодействия сервисов: графовая модель зависимостей на основе distributed tracing (понимание blast radius).
- Предиктивный мониторинг бизнес‑метрик: прогноз продаж и детект отклонений.
- DBA Copilot: помощник по написанию PostgreSQL‑скриптов на базе LLM + библиотеки проверенных скриптов.

Развитие продукта Observability для mission‑critical систем (метрики/логи/трейсы) как платформы данных для AI‑слоя.

- Спроектировал целевой стек для AI‑агентов: tools layer → MCP layer → agents layer → orchestration layer.
- Обеспечиваю полный цикл: discovery → MVP → пилоты → внедрение → метрики использования/качества.

Управление продуктом и командой:
- Стратегия и execution: роадмап, приоритизация, OKR/метрики, CustDev, анализ рынка и вендоров.
